## Manual nudge command

```bash
gh workflow run "Auto-Dispatch Requeue"
```

## Check failure logs

```bash
gh run list --limit 3 --json databaseId,conclusion --jq '[.[] | select(.conclusion=="failure")][0].databaseId' | xargs -I{} gh run view {} --log-failed | grep '##\[error\]' | tail -5
```
